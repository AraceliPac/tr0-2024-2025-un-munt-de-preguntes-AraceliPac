Font-family

Explicació: Defineix la tipografia que s'utilitzarà per al text

Exemple: body { font-family: "Arial", sans-serif; }

Text-align

Explicació: Alinea el text a l'esquerra, la dreta, el centre o justificat

Exemple: body { text-align: right; }

Word-spacing

Explicació: Controla l'espai entre paraules

Exemple: body { word-spacing: 10px; }

Text-shadow

Explicació: Aplica ombres al text

Exemple: body { text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.5); }

Text-decoration

Explicació: Afegeix decoracions al text

Exemple: body { text-decoration: underline; }
